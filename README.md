
# AI Workflow Builder

Low-code canvas to design and run AI workflows.

## Stack
- React (Vite, TypeScript, React Flow)
- Node.js + Express + Prisma
- PostgreSQL (Docker Compose)

## Dev Quickstart
# 1) Start DB
docker compose up -d

# 2) Start API
cd server
npm run dev

# 3) Start Web
cd ../client
npm run dev

Open http://localhost:5173
